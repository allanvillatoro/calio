# Nota sobre URLs de Imágenes

El archivo `data/products.json` actualmente contiene URLs de ejemplo de Unsplash.

Cuando estés listo para usar tus propias imágenes:

1. Crea una cuenta en [Cloudinary](https://cloudinary.com) (gratis)
2. Sube tus imágenes desde la interfaz web de Cloudinary
3. Copia las URLs que Cloudinary te proporciona
4. Reemplaza las URLs en `data/products.json` con las URLs de Cloudinary

**Ejemplo de URL de Cloudinary:**
```
https://res.cloudinary.com/tu-cloud-name/image/upload/v1234567890/tu-imagen.jpg
```

Ver `DEPLOYMENT.md` para instrucciones detalladas paso a paso.
